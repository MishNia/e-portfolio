Harry Potter Pop Quiz

https://mishnia.github.io/hpquiz/

I opted to make a Harry Potter themed quiz, because it’s one of my favorite book and movie series. I added a background image of Hogwarts and Ron’s flying car, which helps set the mood and context of the quiz. The adjusted color scheme features dark backgrounds with light text, which improves readability and reduces eye strain, making it easier for users to engage with the content over longer periods.


Taylor Swift Drag n Drop Quiz

I made a taylor swift themed drag and drop quiz where the users are expected to match the songs to a specific album. I used a muted, Swift-inspired color palette, with light shades for the album boxes and accents in a subtle rose (#d5a6bd), aligning with the theme’s elegance.
